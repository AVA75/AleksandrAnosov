Блок навигации
  • Ширина 15еmc
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом, достаточным, что бы по высоте он начинал захватывать вторую статью, как показано в данном примере
Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фонас
Во всех "статьях" сделайте отступ снизу от соседней равным .2em при помощи свойства margin-
bottom.

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фонас
Во всех "статьях" сделайте отступ снизу от соседней равным .2em при помощи свойства margin-
bottom. Картинка вставлена при помощи тега, плавание организовано пл правому краю внутри
данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
