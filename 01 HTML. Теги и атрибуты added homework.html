Теги форматирования машинного языка

При выполнении програмного кода на языке PERL:

#!/usr/local/bin/perl
print "Content‐type: text/html\n\n";
print "< H1 >Hello World < /h1 >\n";

Программа выведет:

Hello, World!

Однако в браузере у нас будет отображаться результат выполнения этой программы:

Hello, World!
